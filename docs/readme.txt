Chute Da Sorte

No Chute da Sorte o jogador tenta adivinhar uma determinada 
carta que o jogo selecionar anteriormente. A partir dos 
acertos de  naipe e valor da carta, sao computados os pontos.

Para acessar o jogo, va ate a pasta do jogo (via terminal) 
e execute o comando "java -jar ChuteDaSorte.jar".
